# SimpleBackend
利用PHP與HMTL，模擬簡易的補習班資料系統。
大二的資料庫課作業，使用XAMPP建立Apache網頁伺服器。達到新增、修改、刪除、更新功能

簡介:一開始可以新增課程，接著能夠新增學生的資料，也能刪除學生資料，最後可以查詢現在有哪些學生及他們的個人資料與修了哪些課程。
